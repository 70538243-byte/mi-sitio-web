Ejemplo de menú con position: relative

El menú está posicionado de forma relativa. Esto significa que mantiene su lugar en el flujo normal del documento, pero puedes posicionar elementos hijos absolutamente dentro de él.

En este ejemplo, el subrayado verde aparece justo debajo del enlace al hacer hover, usando un pseudo-elemento absolutamente posicionado dentro de cada link.
