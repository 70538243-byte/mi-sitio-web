Dominios, Hosting y Publicación Web

📌 Dominio

Es el nombre único que identifica tu página web en internet.

Ejemplo: www.tiendaonline.com o www.colegio.edu.pe

📌 Hosting

Es el servidor donde se guardan los archivos de tu sitio web (HTML, imágenes, videos, base de datos).

Ejemplo: Tu dominio apunta al hosting para mostrar tu página a todo el mundo.

📌 Publicación en Hosting

Es el proceso de subir tus archivos web (HTML, CSS, JS) al servidor de hosting.

Ejemplo: Usar cPanel para cargar tu carpeta index.html.

📌 Herramientas FTP

Son programas que permiten transferir archivos desde tu computadora al hosting.

Ejemplo: Usar FileZilla para arrastrar archivos y subirlos al servidor.

📌 Publicación en GitHub

Permite subir tu proyecto gratis a internet usando GitHub Pages.

Ejemplo: Crear un repositorio en GitHub y activar Pages para que tu web se vea en: https://tuusuario.github.io/tu-proyecto
