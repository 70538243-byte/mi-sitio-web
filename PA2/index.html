1. Publicación, Bordes, Sombras e Íconos Web

  • Dominio: Es el nombre único de una página web (ejemplo: www.miempresa.com).
  • Hosting: Es el servidor que almacena los archivos de tu sitio.
  • Publicación en Hosting: Subir la web al servidor (por ejemplo, con cPanel).
  • Herramientas FTP: Programas como FileZilla que transfieren archivos.
  • GitHub Pages: Publica proyectos en la nube de forma gratuita.
